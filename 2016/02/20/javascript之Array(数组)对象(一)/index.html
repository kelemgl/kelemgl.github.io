JavaScript之Array（数组）对象（一）

发表于 2016-02-20 | 分类于 JavaScript笔记

请不要忽视基础哦，往往你在项目当中进坑的地方，都是源于你对某些基础知识掌握的不牢靠。

一.创建数组

1.var empty = [];//没有元素的数组
2.var primes = [1,2,3,4,5];//有五个数值的数组
3.var misc = [1.1,true,"a", ];//3个不同类型的元素和结尾的逗号

数组直接量中的值不一定要是常量，他们可以是任意表达式；

它可以包含对象直接量或其他数组直接量；

如果省略数组直接量中的某个值，省略的元素将被赋予undefined值。

数组直接量的语法是允许有可选的结尾的逗号，所以[,,]只有两个元素而非三个。

创建数组的另一种方法，调用构造函数。可以有三种方式调用构造函数。

二.数组元素的读和写

数组是通过[]操作符来访问数组中的一个元素，也可以创建一个元素。

数组是对象的特殊形式。使用方括号访问数组元素就像用方括号访问对象的属性一样。

三.数组长度

每个数组有一个length属性，代表数组的个数。

有一个特殊的行为是设置length属性为一个小于当前长度的非负整数n时,当前数组中那些大于索引值将被删除。

也可以将数组的length属性值设置为大于当前的长度。实际上这不会向数组中添加新的元素，它只是在数组尾部创建一个空的区域。
