语法说明：
eval()函数可计算某个字符串，并执行其中的JavaScript。
如果参数是一个表达式，eval()函数将执行表达式。如果参数是JavaScript语句，eval()将执行JavaScript语句。
string：必须要计算的字符串，其中含有要计算的JavaScript表达式或要执行的语句。

注：加上圆括号的目的：是迫使eval函数，在处理JavaScript代码的时候，强制将括号内的表达式转化为对象而不是作为语句来执行
